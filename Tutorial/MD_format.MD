Markdown 是一种轻量级的标记语言，它的核心理念是“易读易写”。即使是在纯文本状态下，它的排版逻辑也非常直观。

以下是您需要的 Markdown 基础语法及进阶操作指南：

1. 基础语法速览
标题： 使用 # 号，一级标题一个 #，二级标题两个 ##，以此类推（最高支持六级）。

加粗： 在文字两端各加两个星号，如 加粗文字。

斜体： 在文字两端各加一个星号，如 *斜体文字*。

列表：

无序列表：使用 *、+ 或 - 加空格。

有序列表：使用数字加点 1.  加空格。

2. 如何插入图片
在 Markdown 中插入图片的标准语法是：
![图片描述](图片链接或路径)

根据图片来源的不同，有以下几种常用方式：

网络图片： 直接填入图片的 URL 地址。
![风景照](https://example.com/image.jpg)

本地图片： 填入文件在电脑里的相对路径或绝对路径。
![我的照片](./images/photo.png)

快捷粘贴（推荐）：
如果你使用的是 Typora、Obsidian 或 VS Code 等编辑器，通常可以直接将图片文件拖入编辑器，或者截图后直接 Ctrl+V。编辑器会自动帮你生成上述语法，并将图片保存到指定的文件夹中。

3. 如何引用 (Blockquotes)
引用一段话时，只需在行首添加一个右尖括号 >：

这是一段引用文字。
它可以跨行，只需要每行开头都加上 >。

甚至可以嵌套引用。

4. 如何画表格
Markdown 的表格使用竖线 | 来分隔单元格，使用短横线 - 来分隔表头和其他行。

代码示例：

Markdown
| 软件名称 | 主要用途 | 推荐指数 |
| :--- | :---: | ---: |
| Typora | 实时预览写作 | ★★★★★ |
| Obsidian | 知识库管理 | ★★★★☆ |
| VS Code | 通用开发/MD | ★★★★☆ |
语法说明：

冒号的位置决定对齐方式：

:--- 左对齐（默认）

:---: 居中对齐

---: 右对齐

5. 其他常用小技巧
代码块： 使用三个反引号 ``` 包裹代码。

在 Markdown 中，显示代码段主要有两种方式：行内代码和代码块。

1. 行内代码
如果你只想在一段文字中间插入一小句代码，使用单个反引号（即键盘上 Esc 键下方的那个键）包裹。

语法： `代码内容`

效果： 使用 print("Hello World") 来打印。

2. 代码块（多行代码）
对于长篇的代码段，使用三个反引号（也叫“围栏”语法）来包裹。

代码从哪里开始，到哪里结束？
开始： 在代码的首行上方，输入三个反引号 ```。

结束： 在代码的末行下方，输入三个反引号 ```。

超链接： [显示文字](URL地址)。

水平分割线： 使用三个以上的星号 *** 或横线 ---。

Markdown 的魅力在于你可以完全脱离鼠标，只靠键盘就完成精美的排版。建议先从简单的标题和列表开始尝试。
